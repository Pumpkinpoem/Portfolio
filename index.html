About Me

A Cup of Fiction/h3>

This project is a dynamic social media platform designed specifically for book clubs. It allows users to connect, share recommendations, and discuss their favorite reads in a vibrant community setting. As the lead developer, I designed and implemented the front-end using React, creating a user-friendly interface that enhances the user experience. The back-end was built with Django REST Framework, ensuring seamless data management and retrieval. Additionally, I integrated PostgreSQL for robust data storage and utilized Cloudinary for efficient media storage, enabling users to upload and share images effortlessly.

One Dice

this website is a geek café where tabletop games and delightful beverages create a cozy atmosphere. The website features a fully responsive design with a navigation bar, a captivating landing page, and a menu section for drinks and snacks. Users can sign up for the café's email list, choosing a welcome gift of dice sets. The project includes a confirmation page and social media links in the footer. Future enhancements will introduce a gallery page and quiz form.

Tolkien Quizz

Tolkien Quiz is an engaging trivia platform where users can test their knowledge of J.R.R. Tolkien's works, like The Lord of the Rings and The Hobbit. The application boasts a responsive design that allows for seamless use on any device, featuring a navigation bar, score tracking, and the ability to provide feedback. It utilizes HTML, CSS, and JavaScript for development and is deployed on GitHub Pages, ensuring accessibility for Tolkien enthusiasts to challenge themselves and learn more about Middle-earth.

Dungeon Time

Fantasy Adventure Game is a classic text-based adventure implemented in Python, where players explore a dark dungeon filled with traps and monsters, ultimately aiming to defeat a dragon and rescue a princess. The game features a Player class to manage attributes like health points and game progress. Players can make choices that affect their journey through various challenges, including battling monsters and finding a magical sword. Future updates may include an expanded game length, improved combat mechanics, and magical features.

Glad Tidings Times

this is a full-stack web application designed to share uplifting and heartwarming news stories. Built with Django, PostgreSQL, and Cloudinary, it allows users to create accounts, post stories, comment, and vote on articles. Following Agile principles, the project utilized a Kanban board for task management. Key features include user authentication, CRUD operations for posts, and responsive design for accessibility across devices. Automated tests verify model and view functionality, while manual testing ensures a smooth user experience.
